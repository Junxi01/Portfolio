Validation & Accessibility

  1. Open W3C HTML Validator, validate each page by URL or file upload, and save a .jpg screenshot per page as validation/valid-html-*.jpg.
  2. Open W3C CSS Validator, validate styles.css, and save a screenshot as validation/valid-css-global.jpg.
  3. Run AChecker or WAVE on at least the home page. Save a screenshot as validation/valid-accessibility-index.jpg.
  4. Commit the screenshots to the validation/ folder before submission.

Note: If you intentionally ignore certain warnings, document why in readme.txt.
